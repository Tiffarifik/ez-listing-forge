How It Works

From Folder to Storefront

Your designs are organized in folders. EZ Listing Forge turns that structure into published listings automatically.

01

Organize

Drop your design files into listing folders. Each folder becomes one listing. Add a downloads subfolder with your product files.

02

Scan

Hit Scan Folder. EZ Listing Forge reads every listing folder, validates images, and shows you what's ready to go.

03

Generate

AI generates optimized titles, 13 tags, and descriptions tailored for search. Preview everything or let Auto Mode handle it.

04

Publish

Send to draft with one click — or one button press in Auto Mode. Switch between shops from the sidebar and repeat.

API Integration

Built on the Etsy Open API v3

EZ Listing Forge integrates directly with the Etsy Open API v3 using OAuth 2.0 with PKCE for secure authentication. Here's how the app uses the API.

Create Draft Listings

Uses the createDraftListing endpoint to create new listings as drafts. Listings are never auto-published — sellers always review and publish manually from their shop.

Upload Listing Images

Uses the uploadListingImage endpoint to attach product images, branding graphics, and WYR mockups to each listing. AI-generated alt text is included with each image upload for accessibility.

Update Listing Metadata

Uses the updateListing endpoint to push updated titles, tags, descriptions, and pricing to existing listings during bulk editing.

Fetch Existing Listings

Uses the getListingsByShop endpoint to import existing listings for review, editing, and bulk updates.

OAuth 2.0 Authentication

Sellers securely connect their shop via OAuth 2.0 with PKCE. The app never sees or stores login credentials — all authentication is handled through Etsy's authorization page.

Upload Listing Video

Uses the uploadListingVideo endpoint to attach a product video to a listing. Videos are generated locally by the app before uploading.

SKU Generation

The app generates a unique SKU for each listing and includes it when creating or updating listings via the API.

AI Metadata (Local)

Title, tag, and description generation is handled locally via AI — this does not use the Etsy API. Only the final metadata is sent to Etsy when creating or updating a listing.
